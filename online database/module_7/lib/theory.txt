1. Explain what a RESTful API is and its importance in mobile applications.
-> A RESTful API (Representational State Transfer API) is a web service that follows REST (Representational State Transfer) principles to allow communication between clients (e.g., mobile apps) and servers.

2. Describe how JSON data is parsed and used in Flutter.
-> JSON (JavaScript Object Notation) is a lightweight data format used for communication between a Flutter app and a backend server. In Flutter, JSON data is typically fetched from a REST API and converted into Dart objects for easy manipulation.

3. Explain the purpose of HTTP methods (GET, POST, PUT, DELETE) and when to use each.
-> GET	- Retrieve data from the server
   POST	- Send new data to the server
   PUT - Update an existing resource
   DELETE - Remove a resource from the server
