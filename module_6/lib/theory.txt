1.Explain the structure and purpose of forms in Flutter
->Forms in Flutter are used to collect and validate user input. The Form widget provides a container for grouping multiple form fields, handling validation, and managing form states efficiently. It is commonly used for login screens, registration pages, and other input-heavy UI elements.
->Structure of Forms in Flutter : Form Widget,FormField Widgets,GlobalKey<FormState>,Submit Button

2.Describe how controllers and listeners are used to manage form input.
->controllers and listeners are used to manage form input dynamically. They allow developers to capture, manipulate, and respond to changes in user input fields efficiently.
->Controllers : A controller, typically an instance of TextEditingController.
->Listener : A listener is a function that runs whenever the value of a TextEditingController changes. It allows real-time tracking of user input.

3.List some common form validation techniques and provide examples.
->Form validation ensures that user input is correct before processing it. Flutter provides different techniques for validation, including built-in validators, custom functions, and third-party libraries.
->email validation,password strength validation,confirm password validation, phone number validation, etc.
